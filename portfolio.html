HBS Alumni Website: Collborated with designers and engineers to translate high fidelity design comps to webpages. Seating Chart Application: The is auguably one of my favorite projects to date. Developed for HBS, JSON data is transformed by jQuery and rendered with HTML and CSS to render seating chart for use in classes. It is simple and robust application. Notable features inlcude real time rendering and updating of data, multi functional jQuery tool tips and online reports generated with HTML/CSS.

Printable Program Schedule: This simple application leverages JSON data, HTML, CSS and jQuery to generate program schedule. CSS styles dynamically render varying time blocks and color code sessions within a program. The challenge lay in mapping time blocks to divs representing sessions within a day. The container is optimized to fit within standard browser size to minimize scrolling. The application also providing printable schedules for the day or week.
